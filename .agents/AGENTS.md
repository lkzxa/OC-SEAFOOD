# Session Summary Rule

- **Session Summary**: Khi hoàn thành một task hoặc kết thúc một phiên làm việc, bắt buộc phải tạo/cập nhật một file markdown (ví dụ `walkthrough.md` hoặc `session_log.md` trong thư mục `reports/` hoặc thư mục gốc dự án) để ghi lại chi tiết tất cả các hành động, thay đổi đã thực hiện trong phiên làm việc đó.
